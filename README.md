# Nepal-Trekking-Risk-Dashboard

Nepal Trekking Risk Dashboard | Data Visualization Project

In this project, I transformed raw trekking and weather data into meaningful visual insights using data analytics and visualization techniques.

📊 Key Insights from the Dashboard
- Total Bookings, Cancellations and Completion rate
- Geographic risk distribution across Nepal districts
- Weather conditions vs Cancellation rate analysis
- Seasonal trekking trends year-on-year comparison
- Atmospheric pressure impact on cancellation risk
- District-wise cancellation rate heatmap
- Monthly trekking performance CY vs PY

🛠 Tools & Technologies Used
- Tableau Desktop Public Edition
- Python
- Excel

This project helped me understand how data visualization can transform complex tourism and weather data into actionable insights and support data-driven decisions for trekking operators and the Nepal Tourism Board.
